# Portfolio — Jose Luis

Portfolio personal construido con HTML, CSS y JavaScript (sin frameworks).
Objetivo: mostrar proyectos, habilidades y enlaces de contacto.

## Demo
- Web: https://socjoseluis.com *(temporal, cambiar cuando esté publicado en GitHub Pages)*



## Tecnologías
- HTML5
- CSS3 (responsive con Flexbox/Grid)
- JavaScript (DOM + pequeñas interacciones)

## Cómo ejecutarlo en local
Opción 1: abrir `index.html` en el navegador.

Opción 2 (recomendado): usar Live Server en VS Code:
1. Instala la extensión **Live Server**
2. Clic derecho en `index.html` → **Open with Live Server**

## Deploy
Este proyecto está pensado para desplegarse como web estática (por ejemplo con GitHub Pages).
